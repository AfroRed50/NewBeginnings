## Cusom CSS Classes
```
1) .container
    - Centers content for smaller viewports

2) .main-header
    - Sets background color of the header and centers text

3) .main-footer
    - Sets background color of the footer and centers text

4) .center
    - Centers the image on smaller screens

5) .clearfix
    - Fixes location issues of elements when floats are applied

```
